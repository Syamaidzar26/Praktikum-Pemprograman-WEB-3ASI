Praktek Pemprograman Web 1

Sesi 1

Praktek Pemprograman Web 1 akan membahas tentang cara menampilkan halaman web dengan meggunakan HTML, CSS dan Javascript. Mahasiswa diharapkan mampu memahami sintaks dasar dalam menampilkan halaman web, dan dengan emahaman tersebut kemudian menuliskan hasil kreasinya.

HTML

  1. Pengenalan HTML
  2. Tabel
  3. Form

CSS

Cascading Style Sheet (CSS) merupakan standar untuk mengatur tampilan yang ada pada suatu berkas HTML. CSS bukan Bahasa pemprograman tetapi merpakan bahasa deklaratif yang dinyatakan langsung untuk menyatakan suatu tampilan, tidak terdapat logika, percabangan, ataupun perulangan

JasaScript

Javascript merupakan bahasa pemprograman yang berjalan disisi browser, disisi pengguna, atau peramban bukan diserver
